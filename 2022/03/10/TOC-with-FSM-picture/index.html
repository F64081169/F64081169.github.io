(TOC)透過 flask 中的 send_file 拿到fsm狀態圖

需求

靠北我只有mac可以安裝pygraphviz

  1. 需安裝 pygraphviz,request
  2. 連上server(heroku or ngrok)
  3. pipenv等相關套件安裝

程式畫面

步驟

  1. 建立get_file.py，並打入程式碼
  2. 在終端機輸入 pipenv run app.py
  3. 打開另一個終端機輸入pipenv run get_file.py
  4. 即可拿到生成的fsm圖
